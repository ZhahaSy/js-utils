# js-utils

``This is a library of Javascript tools，I created it for the purpose of summarizing the common methods I used to work, so as to avoid being lost when I use it in the future。``

- type: 数据类型检查

- reg: 正则校验

- env：环境检测

- dom: dom操作

- bom：bom操作

- file: 文件操作

- str：字符串操作

- Data Processing 集合操作（数组，对象）

- func ： 函数包装操作
